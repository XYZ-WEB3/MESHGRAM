# Планы развития

## Расширение медиа
- Фото → описание + OCR.
- Документы → извлечение текста.
- Видео → ключевые кадры и сжатое описание.
- Голосовые → транскрипция через Whisper/Vosk.

## LoRa ASCII-video mode
- Экспериментальный режим, который стримит кадры с низким разрешением через mesh.

## Автоперевод
- Автоматический перевод входящих EN/RU сообщений в предпочтительный язык пользователя.

## Анализ важности
- Классификация сообщений по ключевым словам, пометка HIGH PRIORITY.

## Управление устройством
- Команды `/loc`, `/battery`, `/ping` для диагностики Meshtastic-устройства.

## GUI
- Кроссплатформенное приложение (Qt5/PySide или Electron + Python backend) с визуальным мониторингом сети.
