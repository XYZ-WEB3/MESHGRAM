# Протокол общения с LoRa

## Формат пакетов
- Базовая полезная нагрузка — UTF-8 текст до 200 символов.
- Структура исходящего сообщения: `#ID текст`.
- Команда определяется по префиксу `/` и отправляется в неизменном виде.
- Для служебных сообщений (статус, ACK) используется префикс `!`.

## Пример
```
#3 Вода закончилась к вечеру
#3 /status
```

## Ответы
- Meshtastic-пакет без префикса считается свободным текстом и будет переслан в Telegram собеседнику, связанному с `#ID`.
- Команды `/msg`, `/ulist`, `/status` обрабатываются Command Module и отвечают текстом в LoRa.

## Ограничения
- Для уменьшения коллизий Meshtastic Module применяет троттлинг (по умолчанию 1.5 сек между пакетами).
- Все пакеты логируются и могут быть повторно отправлены при сбое.
