# Команды управления

| Команда | Канал | Описание |
| --- | --- | --- |
| `/ulist` | LoRa, Telegram | Список активных пользователей с их ID. |
| `/history #ID` | LoRa | (план) Возвращает 10 последних сообщений пользователя. |
| `/msg #ID текст` | LoRa | Принудительная отправка сообщения от оператора. |
| `/status` | LoRa | Состояние очередей и каналов. |
| `/net` | LoRa | Планируемое расширение с информацией о Telegram и LoRa ошибках. |
| `/mesh` | LoRa | Вывод параметров Meshtastic. |
| `/flush` | LoRa | Очистить исходящую очередь. |
| `/help` | LoRa | Краткая подсказка. |

Поддержка новых команд выполняется через регистрацию обработчиков в `CommandHandler`. Каждая команда получает аргументы (`list[str]`) и возвращает строку, которая отправляется обратно через Meshtastic.
