# Протокол общения с LoRa

## Формат текстовых пакетов
```
#<ID> <payload>
```
- `#<ID>` — идентификатор пользователя (например `#3`).
- `<payload>` — UTF-8 текст, до 200 символов (рекомендуется).
- Пакеты команд начинаются с `/` и не содержат ID.

## Подтверждения
- Meshtastic не даёт гарантированных ACK, поэтому сервис повторно отправляет сообщения при истечении `retry_delay`.
- При получении команд ответ немедленно отправляется тем же отправителем.

## Throttling
- Между пакетами соблюдается `throttle_seconds` (по умолчанию 2.5 сек).
- Дополнительно возможно динамическое увеличение задержки при росте `retries` (запланировано).

## Состояние
- Команда `/mesh` (план) будет возвращать RSSI, SNR и заряд батареи.
