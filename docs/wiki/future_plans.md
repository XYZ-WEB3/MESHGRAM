# Планы развития

1. **Медиа → текст** — интеграция OCR, распознавание речи (Whisper/Vosk), извлечение текста из документов.
2. **ASCII-video режим** — экспериментальная передача кадров по LoRa.
3. **Автоперевод** — автоопределение языка и перевод входящих сообщений.
4. **Анализ важности** — классификация сообщений по критериям "срочно/норма".
5. **Команды устройства** — `/loc`, `/battery`, `/ping`, `/mesh` для мониторинга железа.
6. **GUI-приложение** — десктопный клиент на Qt или Electron + Python backend.
7. **Расширенный мониторинг** — метрики Prometheus, уведомления о сбоях.
